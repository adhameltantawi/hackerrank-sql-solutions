# SQL Solutions & Data Analytics

A comprehensive repository showcasing robust, optimized SQL solutions and practical data analysis exercises. This project demonstrates proficiency in querying, manipulating, and extracting actionable insights from complex datasets across different relational database environments.

## Overview

This repository serves as a portfolio of SQL challenges from top technical platforms, ranging from foundational data retrieval to advanced analytical queries. Each solution focuses on writing clean, efficient, and well-structured code that adheres to industry best practices and emphasizes query optimization.

## Key Competencies & Topics Covered

- **Basic Queries:** Data extraction (`SELECT`), robust filtering (`WHERE`), conditional logic (`CASE WHEN`), and handling `NULL` values.
- **Joins:** Complex relational mapping using `INNER`, `LEFT`, `RIGHT`, `FULL OUTER` joins, and self-joins to consolidate data across normalized schemas.
- **Aggregations:** Grouping data (`GROUP BY`, `HAVING`) and utilizing core aggregate functions (`SUM`, `AVG`, `COUNT`, `MIN`, `MAX`) for insightful summary statistics.
- **Subqueries:** Implementing dynamic filtering via nested queries, correlated subqueries, derived tables, and optimal use of `EXISTS` vs `IN`.
- **Window Functions:** Performing sophisticated analytics with `ROW_NUMBER()`, `RANK()`, `DENSE_RANK()`, `LEAD()`, `LAG()`, and `PARTITION BY` clauses to solve complex ranking, pagination, and running-total problems.
- **Advanced Database Concepts:** Structuring massive queries using Common Table Expressions (CTEs) for maintainability, query execution flow, and advanced string/date manipulations.
